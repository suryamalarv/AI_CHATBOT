# Employee Helpdesk Chatbot
A simple Flask-based chatbot that answers question about HR, IT and Event queries.

# Features
Handles HR, IT, and Event queries using local text files.

Filters bad language.

Lightweight, responds in < 5 seconds.

Easily extendable to use NLP / deep learning models.
